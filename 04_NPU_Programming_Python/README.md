## overview
Module-LLM + pyaxengineで、画像認識を行うサンプルです。<br>


## preparation

必要なパッケージのインストール

```
root@m5stack-LLM # pip install -r requirements.txt
root@m5stack-LLM # pip install axengine-0.1.0-py3-none-any.whl
```
